# BootRoot: Root of No Return
**Author: ThinhDe**

**qemu-system-i386 -m 512 -hda V1t_win2k_disk.qcow2**


On a scorching summer day in 2006, the computer of a college student named Tùng Núi suddenly died completely. No matter how many times he tried reinstalling Windows, nothing could bring it back to life.

All that stared back at him was a red screen displaying a taunting message: “tìm tới và chan bố may đê”

Unfortunately, although the forensic team arrived, they were unable to recover the study materials from Cô Giáo Thảo that Tùng had stored on the machine. However, they did manage to recover a file named eEyeBootRoot2005.exe, and it appears that this file had been executed just before the computer was permanently bricked.
